# 🎯 Interview Mirror – AI-Powered Interview Performance Analyzer

## 📌 Overview
Interview Mirror is an AI-based web application that analyzes a candidate’s eye contact, posture, stability, and confidence during mock interviews using computer vision and MediaPipe.

---

## 🚀 Features
- Real-time webcam analysis
- Eye contact detection
- Posture analysis
- Movement stability tracking
- Confidence scoring system
- Personalized feedback report

---

## 🧠 Tech Stack
- Python
- Streamlit
- OpenCV
- MediaPipe
- NumPy

---

## 📊 How it works
1. User starts the application
2. Webcam captures live video
3. MediaPipe detects facial landmarks
4. System analyzes:
   - Eye contact
   - Posture
   - Stability
5. Confidence score is generated
6. Final feedback report is displayed
